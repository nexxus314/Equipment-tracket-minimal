## Equipment Tracker

### How to run
1. Clone the repo
2. Backend:
   - cd backend
   - npm install
   - node server.js
3. Frontend:
   - cd frontend
   - npm install
   - npm start
### Added features
- Add search and filters
- Better UI styling
### Assumptions
- Used a JSON file as database for simplicity
- Single-user system (no auth)
